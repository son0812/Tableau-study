##CH1. 태블로 이해하기

### 1. 디지털로의 전환
- DT : 디지털트랜스포메이션(기업 당면 과제)
→ 산재되어 있는 데이터를 집중화 시키고 비즈니스에 적극적으로 활용할 수 있도록 하고 데이터에 대한 내부 문화가 바뀌두록 유도
- DL : 데이터 리터리시 (기업 구성원의 당면 과제)
→ 데이터를 보고 활용할 수 있는 능력
- DV : 데이터 비주얼리제이션 (데이터 리터리시를 강화하는 방법)
→ 데이터를 시각적으로 표현하고 찾은 인사이트를 조직 내 공유 및 협업하는 과정

- 탐색을 통해 이해하고 이를 다른 사람과 커뮤니케이션, 협업을 통해 인사이트를 도출하고 공유함
- 태블로를 도입하기 전에는 SAP 데이터 다운 후 엑셀에서 가공하고 시각화하였는데, 이제는 SAP 데이터를 태블로에서 바로 연결하고 실시간으로 데이터의 흐름 추적 가능
- Tableau Desktop 에서 시각적 분석 후 Tableau Server로 공유
- 데이터 시각화 : 데이터에 색상과 모양 그리고 여러 요소를 배치해서 데이터를 시각적으로 표현한 것
- 데이터 시각적 분석 : 데이터 시각화를 통해 스토리텔링 및 인사이트를 찾는 분석 기법
----
### 2. 태블로 이해 및 설치하기
- 태블로란? : 데이터를 분석 및 시각화하는 BI(Business Intelligence) 솔루션
- BI : 비즈니스를 운영하면서 얻은 데이터를 수집, 저장, 분석하여 성과를 최적화하는 프로세스와 방법을 망라하는 포괄적 용어 (데이터를 통해 의사결정을 돕고 비즈니스 목표를 달성하게 유도)
- 태블로는 드래그 앤 드롭만으로도 시각화 가능 + 데이터 원본에 없는 필드 생성 가능 (엑셀과 유사한 함수) + 모바일 대시보드도 가능
- csv 형태의 파일은 `텍스트파일`로 연결
- 태블로 아이콘을 누르면 작업창이 뜨고 `데이터에 연결`을 선택 혹은 db모양의 아이콘을 선택하면 데이터 선택 가능
- 데이터 그리드에서는 원본데이터를 미리보기 가능
